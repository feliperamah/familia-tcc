# Tudo Começa em Casa

Site estático gerado a partir do Claude Design.

## Estrutura
- `index.html` — página da VSL (vídeo). Fica em `seudominio.com`.
- `curso-familia-tcc.html` — página de vendas. Fica em `seudominio.com/curso-familia-tcc.html`.

O botão da VSL redireciona automaticamente para a página de vendas ao final do vídeo.

## Deploy
Este projeto pode ser hospedado em qualquer serviço de hospedagem estática (Hostinger, Vercel, Netlify, GitHub Pages), bastando apontar o "diretório raiz" / "output" para a pasta que contém os arquivos, sem comando de build.
