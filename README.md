# Meu Site

Site estático gerado a partir do Claude Design.

## Estrutura
- `index.html` — página única e autossuficiente (os recursos são carregados no próprio navegador).

## Deploy
Este projeto pode ser hospedado em qualquer serviço de hospedagem estática (Hostinger, Vercel, Netlify, GitHub Pages), bastando apontar o "diretório raiz" / "output" para a pasta que contém o `index.html`, sem comando de build.
